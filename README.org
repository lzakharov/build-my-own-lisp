* Build My Own Lisp
  Learning C and building my own programming language with the book
  [[http://www.buildyourownlisp.com]["Build Your Own Lisp"]].
 
** Building
   Just run:
   #+BEGIN_SRC shell 
   make
   #+END_SRC
   
** Running
   After building type:
   #+BEGIN_SRC shell
   ./lispy
   #+END_SRC
   
** Examples
   #+BEGIN_SRC
   lispy> + (* 2 3) 4 
   10
   lispy> head (tail {1 2 3 4})
   {2}
   lispy> def {x} 123
   ()
   lispy> x
   123
   lispy> \ {x y} {+ x y}
   (\ {x y} {+ x y})
   lispy> fun {curry f xs} {eval (join (list f) xs)}
   ()
   lispy> curry + {5 6 7}
   18
   #+END_SRC
